Como linkar o CSS com a página HTML Características do css Herança Especificidade Via de regra, o CSS funciona de cima para baixo. Esssa regra é quebrada com a especificidade, que diz que quanto mais específica for a seleção, maior prioridade das proproiedades citadas. Cascata Ordem de prioridade: --> Inline --> interno --> Externo Ou seja, primeiro o navegador lê o CSS externo, depois o interno e por último o inline. Unidades de Medida Medidas absolutas px (pixel) pt (points) --> muito utilizada para fontes in (inchs) cm (centimeters) mm (milimeters) Medidas relativas a fonte em (se baseia no font-size do elemento pai) rem (se baseia no font-size do elemento raíz - tag html - geralmente é de 16px por padrão do navegador) --> root em % (considera a medida do elemento pai como 100%) ex (medida da altura da letra x na fonte que está sendo usada) ch (largura de um caracter de fonte, o problema em utilizar essa unidade é que nem todas as fontes são monospace) viewport (É o espaço visível da tela) --> (vh, vw, vmin, vmax) inherit (usado para forçar herança do elemento pai) currentColor (força herança de cor do elemento pai) Responsividade x Adaptatividade Breakpoints mais utilizados: 320, 480, 760, 960, 1200, 1600 Propriedade object-fit (fill, contain, cover, none, scale-down) Media queries @media dita as regras para o tamanho de tela. Pode ditar mais de uma condição. Como valor, passa-se o nome da animação (animation-name), o tempo de duração (animation-duration), tempo de espera (animation-delay), número de repetições ou infinite (animation-iteration) FAZER UM EXEMPLO DE DIV QUE MOVIMENTA E MUDA O BG-COLOR Sombras Podem ser feitas utilizando a propriedade box-shadow ou text-shadow Box-shadow: distância horizontal, distância vertical, blur, color Text-shadow: (mesmos valores) Transições Utilizada para que a transição entre propriedades do CSS seja feita de forma suave. Funciona como se fosse uma animação, mas é automática, como um controle de transição. Transition: all (tipo de propriedade), tempo de duração e forma (linear, ease-in, ease-in-out) Webkit Engine responsável por aplicar o CSS na tela do navegador. Cada navegador tem seu engine: IE = Trident Edge = EdgeHTML Firefox = Gecko Opera = Webkit Safari = Webkit Chrome = Webkit Word-wrap Quebra a palavra para caber dentro do seu elemento pai, não quebramndo o layout da aplicação Pode ter os valores de normal ou break-word Flexbox CSS Tipo de display que torna o box em que for acionado, flexível. Propriedades do CSS: display: flex; flex-direction (por padrão é row, pode ser column, row-reverse, column-reverse) flex-wrap: nowrap, wrap flex-flow: é o flex-wrap + flex-direction (primeiro se especifica a direção e depois o wrap) justify-content: center, flex-start (inicia do início do flex) , flex-end (alinha para o lado do final do flex), space-around, space-between (diz respoeito à flex-direction setada) align-items: alinha para o lado não setado como direção (os valores possíveis são os mesmos que o justify-content) align-content: alinha todo o conteúdo da tag pai, quasndo há wrap Propriedades na tag filha align-self: flex: proporção de tamanho para cada item (configurado dentro do item interno) order: altera a posição do item. A ordem padrão é zero. Seletor pelo atributo: O atributo deve aparecer entre colchetes input[required] input[type=text] input[type=password] input[name='user'] O valor do name deve estar entre aspas, pois é um valor que foge do padrão do sistema, é aleatório, setado pelo programador. Pode-se selecionar também partes de determinado valor de certo atributo. Para isso, usa-se o sinal * antes da igualdade do atributo. a[href*='economia'] Para atributos que comecem com determinado valor, utiliza-se o ^ a[href^='https'] Para valores que finalizem com determinada string, usa-se o $: a[href$='html'] Seletor de elemento vazio (:empty) div:empty Seletor de primeira letra e primeira linha. Usamos os dois dois pontos, por se tratar do conteúdo do elemento, não de um atributo e nem do próprio elemento p::first-letter{} (para a primeira letra) p::first-line{} Seletor de primeiro ou último elemento. Dessa vez utilizamos apenas um dois pontos, pois se trata de elementos em si. div:firts-child{} div:last-child{} Seletor negativo: seleciona os elementos que não possuem as características especificadas :not(seletor dos itens a não estilizar){} Seletor NTH. Item específico dentre vários itens alinhados. div:nth-child(4){} div:nth-last-child(2){} Seletor ::before/::after. Usados para criar umpseudoconteúdo em formato de elemento anter ou dempois do conteúdo atual do elemento. Geralmente são acompanhados pela propriedade content. div::before{ content:"..."; } RGBA rgb com alpha (taxa de anti-transparência da cor, que vai de 0 a 1) rgba(255,215,150,0.5) Tranform é uma propriedade geral que caracteriza uma transformação de algum elemento scale() -> aumenta a escala de forma homogênea a aprtir do centro do elemento. Quando possui dois valores, o primeiro representa a escala horizontal e o segunda vertical scale(2) scale(2,1) scaleX(3) scaleY(2) rotate() -> Rotaciona o elemento a partir de seu centro A unidade de medida nesse caso é o deg rotate(40deg) rotateX(60deg) skew() -> Distorce o elemento horizontalmente, verticalmente ou ambos. skew(20deg, 30deg) Box-sizing
