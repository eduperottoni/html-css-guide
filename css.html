Como linkar o CSS com a página HTML Características do css Herança Especificidade Via de regra, o CSS funciona de cima para baixo. Esssa regra é quebrada com a especificidade, que diz que quanto mais específica for a seleção, maior prioridade das proproiedades citadas. Cascata Ordem de prioridade: --> Inline --> interno --> Externo Ou seja, primeiro o navegador lê o CSS externo, depois o interno e por último o inline. Unidades de Medida Medidas absolutas px (pixel) pt (points) --> muito utilizada para fontes in (inchs) cm (centimeters) mm (milimeters) Medidas relativas a fonte em (se baseia no font-size do elemento pai) rem (se baseia no font-size do elemento raíz - tag html - geralmente é de 16px por padrão do navegador) --> root em
