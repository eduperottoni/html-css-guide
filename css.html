Como linkar o CSS com a página HTML Características do css Herança Especificidade Via de regra, o CSS funciona de cima para baixo. Esssa regra é quebrada com a especificidade, que diz que quanto mais específica for a seleção, maior prioridade das proproiedades citadas. Cascata Ordem de prioridade: --> Inline --> interno --> Externo Ou seja, primeiro o navegador lê o CSS externo, depois o interno e por último o inline. Unidades de Medida Medidas absolutas px (pixel) pt (points) --> muito utilizada para fontes in (inchs) cm (centimeters) mm (milimeters) Medidas relativas a fonte em (se baseia no font-size do elemento pai) rem (se baseia no font-size do elemento raíz - tag html - geralmente é de 16px por padrão do navegador) --> root em % (considera a medida do elemento pai como 100%) ex (medida da altura da letra x na fonte que está sendo usada) ch (largura de um caracter de fonte, o problema em utilizar essa unidade é que nem todas as fontes são monospace) viewport (É o espaço visível da tela) --> (vh, vw, vmin, vmax) inherit (usado para forçar herança do elemento pai) currentColor (força herança de cor do elemento pai) Responsividade x Adaptatividade Breakpoints mais utilizados: 320, 480, 760, 960, 1200, 1600 Propriedade object-fit (fill, contain, cover, none, scale-down) Media queries @media dita as regras para o tamanho de tela. Pode ditar mais de uma condição. Animações Utiliza-se keyframes (onde são setados os quadros (frames) da animação) Em seguida, coloca-se o nome da animação setada Para alterar as características, usa-se porcentagem para indicar em que momento do tempo de animação elas serão usadas Para chamar a animação, utiliza-se a propriedade CSS animation. Como valor, passa-se o nome da animação (animation-name), o tempo de duração (animation-duration), tempo de espera (animation-delay), número de repetições ou infinite (animation-iteration) FAZER UM EXEMPLO DE DIV QUE MOVIMENTA E MUDA O BG-COLOR Sombras Podem ser feitas utilizando a propriedade box-shadow ou text-shadow Box-shadow: distância horizontal, distância vertical, blur, color Text-shadow: (mesmos valores) Transições Utilizada para que a transição entre propriedades do CSS seja feita de forma suave. Funciona como se fosse uma animação, mas é automática, como um controle de transição. Transition: all (tipo de propriedade), tempo de duração e forma (linear, ease-in, ease-in-out) Webkit Engine responsável por desenhar o CSS na tela do navegador
