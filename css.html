Como linkar o CSS com a página HTML Características do css Herança Especificidade Via de regra, o CSS funciona de cima para baixo. Esssa regra é quebrada com a especificidade, que diz que quanto mais específica for a seleção, maior prioridade das proproiedades citadas. Cascata Ordem de prioridade: --> Inline --> interno --> Externo Ou seja, primeiro o navegador lê o CSS externo, depois o interno e por último o inline. Unidades de Medida Medidas absolutas px (pixel) pt (points) --> muito utilizada para fontes in (inchs) cm (centimeters) mm (milimeters) Medidas relativas a fonte em (se baseia no font-size do elemento pai) rem (se baseia no font-size do elemento raíz - tag html - geralmente é de 16px por padrão do navegador) --> root em % (considera a medida do elemento pai como 100%) ex (medida da altura da letra x na fonte que está sendo usada) ch (largura de um caracter de fonte, o problema em utilizar essa unidade é que nem todas as fontes são monospace) viewport (É o espaço visível da tela) --> (vh, vw, vmin, vmax) inherit (usado para forçar herança do elemento pai) currentColor (força herança de cor do elemento pai) Responsividade x Adaptatividade Breakpoints mais utilizados: 320, 480, 760, 960, 1200, 1600 Propriedade object-fit (fill, contain, cover, none, scale-down)
