      Tag picture É usada para que uma imagem seja selecionada de acordo com regras de responsividade. Dentro dela, utilizamos as tags source e img Tag source Utilizada em conjunto com a tag picture para seleção do src (srcset) de acordo com o tamanho de tela (media) atributo media (nela são setados os media queries) atributo srcset (seleciona o arquivo a ser utilizado) Tag img Utilizada como o padrão para quando as regras que estiverem no atibuto media da tag source não forem estabelecidos. Ex.: data-name, data-lastname. Serve para armazenar algum dado no HTML para ser usado dinamicamente com JS
